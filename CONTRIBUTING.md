# Contributing to QuotaPet

[简体中文](CONTRIBUTING.zh-CN.md)

Thank you for helping improve QuotaPet. Keep changes local-first, small, testable, and compatible with the security and performance contracts in this repository.

## Before opening a pull request

1. Create a focused branch and explain the user-facing reason for the change.
2. Add or update tests before changing behavior.
3. Run `swift test --disable-sandbox`, `./scripts/build-app.sh`, and `./scripts/verify-package.sh`.
4. Update English and Simplified Chinese user-facing text together.
5. Describe privacy, security, performance, dependency, and compatibility impact.

## Provenance and license requirements

For every contribution, record the origin, author, and license of any added or adapted **code**, **font**, **image**, **icon**, **sound**, **screenshot**, or **dependency**. Confirm that the license permits source distribution and public binary redistribution. Do not submit:

- copied product artwork, logos, fonts, or interface assets without explicit rights;
- confidential source, account data, credentials, tokens, private paths, or customer content;
- generated media whose source inputs or redistribution rights cannot be explained;
- a runtime dependency without license, security, privacy, and maintenance review.

The existing Q-shaped mascot and application icon are generated by this repository. Changes to their drawing code must remain original and must not imitate OpenAI or another product's protected artwork.

## Brand and product naming

Keep the product title `QuotaPet`. Third-party product/model names may be used descriptively inside compatibility and quota content, but not in a way that implies affiliation or endorsement. Review [LEGAL.md](LEGAL.md) before changing public names, icons, screenshots, or release metadata.

## Pull-request checklist

- [ ] Tests fail before the behavior fix and pass afterward.
- [ ] No secrets, personal data, private paths, analytics, or unapproved network calls were added.
- [ ] Every new asset and dependency has documented provenance and license.
- [ ] English and Simplified Chinese documentation remain equivalent.
- [ ] Performance remains inside the repository baseline.
