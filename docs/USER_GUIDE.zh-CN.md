# QuotaPet 使用指南

[English](USER_GUIDE.md)

## 第一次打开

从“应用程序”打开 QuotaPet。它是菜单栏应用，所以不会显示 Dock 图标。悬浮头像和菜单栏环形图默认展示“通用使用限额”的剩余量，点击头像可展开完整详情。

如果暂时没有用量，QuotaPet 会展示发现的 Codex 可执行文件。只确认你认识的路径；每次启动前都会重新校验，文件内容或签名发生有效变化后会要求重新确认。

## 日常操作

- 点击悬浮头像：展开详情。
- 点击详情左上角头像：收起详情。
- 拖动任一状态：移动整个窗口。头像始终跟随同一个左上角锚点，窗口完整留在当前屏幕内。
- 点击菜单栏环形图：打开或关闭紧凑用量卡片。
    - 右键菜单栏环形图：刷新、显示或隐藏桌宠、切换连接模式、恢复交互、打开设置、使用说明、关于或退出。
    - 设置中可切换「跟随系统 / 简体中文 / English」，菜单、详情与刷新态会即时切换。
    - 刷新若长时间无结果，会先提示「即将重新连接」，再自动恢复一次。
    - 按 `⌥⌘U`：唤起头像，并关闭鼠标穿透以恢复点击。
    - 点击“立即刷新”：头像先显示加载，再在新数据到达后短暂显示对号。

详情中的两个服务额度名称固定为 **通用使用限额** 和 **GPT-5.3-Codex-Spark 使用限额**。距重置低于 24 小时按小时显示，更长时间按天显示。

## 设置

- **节能模式**是新安装默认值，读取一次后退出 Codex 子进程。
- **实时模式**保留一个经过校验的子进程连接，以更快更新。
- **始终置顶**控制悬浮窗层级。
- **鼠标穿透**会把点击交给后面的应用；按 `⌥⌘U` 或使用“恢复桌宠交互”即可关闭。
- 通知只在本机生成并且默认关闭；登录启动使用 macOS 官方登录项接口。

## 语言

默认跟随 macOS 首选语言。也可在设置中选择 **跟随系统 / 简体中文 / English**；菜单栏、详情卡、刷新态和设置文案会即时切换。未来新增语言只需增加本地化资源并通过目录校验。

## 更新

公开签名版发布后，退出 QuotaPet，从 GitHub Releases 下载更高版本 DMG 或 ZIP，核对公开校验值后替换“应用程序”中的 App；原有偏好会保留。

官方 Homebrew Cask 发布后可执行 `brew upgrade --cask quotapet`。在首个已公证公开版本出现前，请按 README 的命令从源码构建和安装。

## 排查问题

- **用量不可用：**确认官方 Codex 已安装并登录，核对界面展示的可执行文件路径后刷新。
- **数据旧或不对：**点击“立即刷新”。警告状态只保留上一次真实数据，不会编造额度。
- **头像点不了：**按 `⌥⌘U`，或右键菜单栏图标选择“恢复桌宠交互”。
- **快捷键冲突：**改用菜单栏入口；设置页会说明快捷键是否被其他 App 占用。
- **头像不见了：**启用“显示桌宠”并按快捷键；显示器变化后应用会自动把窗口收回屏幕。
- **锁屏查看：**macOS 不保证锁屏时显示第三方菜单栏界面，QuotaPet 不会绕过锁屏安全限制。

## 卸载和重置

先关闭“登录时启动”，退出 QuotaPet，再删除 `/Applications/QuotaPet.app`。偏好默认保留；如需完全重置：

```bash
defaults delete io.github.asazhangyongchao.quotapet
```
