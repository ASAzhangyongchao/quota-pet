# 更新记录

[English](CHANGELOG.md)

重要变更遵循语义化版本规范。

## [Unreleased]

### 调整

- 空闲动画改为**只动脸**：按状态多帧眨眼 / 嘴形微变，整块宠物不再挤压或侧倾。
- 静态表情更拟人：额度低用担忧眼形，开心眨眼带笑意加深，耗尽时嘴部轻「吸气」。
- 约 8–16 秒一次脸部表情；节能模式仍会播这些短促重绘。
- Codex/ChatGPT 更新后仅 inode/哈希变化时不再卡在信任校验；已确认指纹会滚动，信任失败或 App Server 反复退出时会自动重建连接。

### 修复

- 设置里检查更新时，「正在检查…」只出现在按钮上，不再在下方再重复一行。
- 手动刷新时若 Codex 一直不回包，「正在重新连接…」不再永远转圈；二次超时后会失败，按钮可再点。
- ChatGPT 自带 Codex 更新中挂掉时，会自动改用其他已信任的 Codex，并短暂避开坏路径，避免反复撞上正在更新的应用。
- 设置里「Codex 来源」改为人话说明：当前优先用哪个（ChatGPT 应用 / 终端等）、备选待确认、PATH 无效噪音默认折叠。

## [0.1.4] - 2026-07-21

### 新增

- 手动刷新超时先提示再自动重连一次。
- 更宽可滚动的设置页，分组排版；勾选项旁点 `?` 弹出说明。
- 设置显示当前版本，并可手动检查 GitHub Releases（经公开 Atom 源，避免未认证 API 限流）。
- 设置内语言：跟随系统 / 简体中文 / English，界面即时切换。
- 菜单栏「使用说明」「关于」；用量弹层避开遮挡顶栏。

### 调整

- 手动刷新可唤醒已停止的连接，避免只转圈无响应。
- 设置中信任路径可换行、可选中，不再轻易变成「…」。
- 公开版本元数据更新为 0.1.4 build 12。

### 修复

- ChatGPT 更新后用户属主的官方捆绑 Codex 也会自动信任，避免反复点确认。
- `dist/` 构建产物不再被 Spotlight/启动台索引，避免出现两个 QuotaPet。
- 设置中 Codex 信任预览只显示少量优先项；完整列表在可滚动弹窗中查看。
- 使用说明链接改为 `ASAzhangyongchao/quota-pet`。
- 右键菜单打开前按当前设置语言重建文案。
- 设置里选择器标题不再被裁掉首字母。
- 桌宠显示开关正确读取历史数字偏好；展开详情时切换语言即时生效。

## [0.1.3] - 2026-07-20

### 新增

- 为折叠头像和展开玻璃卡片增加静态语义光晕。
- 增加高对比度的已用/剩余环形分段及相同语义的无障碍额度条。
- 增加相互独立的中英文法律、品牌与贡献说明。
- 设置页增加本地化“关于与法律说明”，发布流程增加法律复核门禁。

### 调整

- 可见头像保持 72pt，只在外层增加 6pt 且不越过屏幕的光晕边距。
- 公开版本元数据更新为 0.1.3 build 4。

## [0.1.2] - 2026-07-19

### 新增

- 英文优先的本地化结构、简体中文资源和其他语言回退英文机制。
- 相互独立的中英文使用、发布和更新文档。
- 统一的 `VERSION` 版本源，以及版本和语言资源打包校验。

### 调整

- 两类服务额度改名为“通用使用限额”和“GPT-5.3-Codex-Spark 使用限额”。
- 展开与折叠都以头像左上角为锚点。
- 拖动和尺寸变化后的窗口始终完整留在当前屏幕。

### 修复

- 不再展示后端内部 `primary` 字段和误导性的未知名称说明。
- 菜单、设置、通知、无障碍提示、日期和底层错误完整本地化。

## [0.1.1] - 2026-07-19

- 增加 Codex 本地用量读取、菜单栏环形图、矢量桌宠、玻璃详情、刷新反馈、信任审核、性能门禁与公开发布准备流程。
